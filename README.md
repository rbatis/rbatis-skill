# rbatis-skill

Claude Code skill for Rbatis - Rust high-performance ORM framework.

## Installation

```bash
# 在 Claude Code 中安装
git clone https://github.com/rbatis/rbatis-skill.git ~/.claude/skills/rbatis-skill
```

## Usage

After installation, Claude Code will automatically use this skill when you ask about:
- Rbatis CRUD operations
- html_sql / py_sql dynamic SQL
- Database migrations
- ORM queries
- Transaction management

## Skill Info

- **name**: rbatis
- **description**: Rust high-performance ORM framework - compile-time dynamic SQL, MyBatis compatible syntax
- **repository**: https://github.com/rbatis/rbatis-skill
